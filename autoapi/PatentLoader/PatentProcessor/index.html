class PatentLoader.PatentProcessor.PatentProcessor¶

Class for processing patent data, including reading, cleaning, and transforming patent records.

Methods: - __init__: Initializes the DataFrame attributes and defines the columns and data types. - read_patents: Reads patent data from an Excel file. - load_patents: Loads patent data from a DataFrame. - process_design: Processes each row of the DataFrame to clean and normalize patent data. - start_process: Starts the processing of the patent DataFrame.
